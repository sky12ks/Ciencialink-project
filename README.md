# Página NexBionics - Desarrollo Web

Una página web desarrollada para la venta de protesis medicas

## Descripción

Este proyecto contiene los siguientes archivos:
- **HTML5** Archivo con etiquetas para la estructura
- **JavaScript** Archivo para eventos y modo oscuro
- **Bootstrap** Framework para estilos y diseño responsivo
